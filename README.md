# Nyan Cat 3D

## Descripción de objetivos
Experimentación con videojuegos: implementación de modelos 3D en páginas web interactivas utilizando la biblioteca Three.js.

## Descripción del proyecto
El proyecto consiste en crear un Nyan Cat en 3D dentro de un entorno infinito (similar al espacio 3D de Blender) que pueda ser controlado por el usuario, como un videojuego.

En este entorno aparecerán diferentes ítems que el usuario podrá recoger (por ejemplo, un sombrero mexicano, una versión retro o una versión corrupta). Cada ítem modificará el entorno visual del juego, la paleta de colores, la música y el diseño del propio Nyan Cat.

La página registrará el tiempo que el usuario pasa en ella (por ejemplo: “You’ve NYANED for 1114.8 seconds”) y lo guardará en una base de datos global.
